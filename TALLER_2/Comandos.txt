1. Establecer hostname

en
config t
hostname <name>


2. Establecer password y encriptar

enable
configure terminal
enable password <pass>
exit
show running-config
ctrl +c
configure terminal
service password-encryption
exit
show running-config
copy running-config startup-config
ENTER X2
reload


3. Establecer secret (el encriptado se hace automaticamente)
enable 
configure terminal
enable secret <secret>
exit 
show running-config
ctrl + c
copy running-config startup-config
EXTER X2
reload
